E004 - invalid username (length)
E005 - invalid password (regex)
E006 - invalid email (regex)
E007 - invalid email (conflict)

E008 - invalid or expired JWT token
E009 - internal server error

E010 - invalid password (length)
E011 - malformed request

E012 - login unsuccessful

S005 - username validated
